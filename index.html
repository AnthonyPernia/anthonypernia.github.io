Sobre mi

Soy Ingeniero Electrónico, pero desde el inicio de mi carrera me intereso el desarrollo de software, e intento capacitarme cada vez mas en funcion de ello.

He trabajado con algunos lenguajes tales como Java en Android así como también para JSP, JavaScript en Web y programación para Arduino con librerías como Johnny-Five, aunque también he usado el lenguaje nativo basado en C++, Python para analisis de datos y machine learning con librerías como Pandas, Sklearn, Tensorflow, Numpy, torch, NLTK entre otros, con bases de datos Relacionales como PostgreSQL, MySQL, usando herramientas de Warehousing como Teradata, Hadoop, Azure. En cuanto a Infraestructura fisica, poseo experiencia en redes, en configuración equipos CISCO, HUAWEI y Juniper, usados en Backbone de redes móviles, específicamente en el Packet Core. Actualmente estoy enfocado en el área de Big Data y Machine Learning. Me considero una persona Proactiva, apasionada por la tecnología y mi meta es aprender algo nuevo todos los días.
