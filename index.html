Sobre mí

Soy Ingeniero Electrónico, que desde el inicio de mi carrera me intereso el desarrollo de software, actualmente estoy enfocado en el área de Big Data y Machine Learning.

He trabajado con lenguajes como Java en Android y JSP, JavaScript en Web y programación para Arduino con librerías como Johnny-Five. También he usado el lenguaje nativo basado en C++, Python para analisis de datos y machine learning con librerías como Pandas, Sklearn, Tensorflow, Numpy, torch y NLTK. Para bases de datos relacionales he trabajado con PostgreSQL, MySQL, usando herramientas de Warehousing como Teradata, Hadoop y Azure. En cuanto a Infraestructura fisica, poseo experiencia en redes, en configuración equipos CISCO, HUAWEI y Juniper, usados en Backbone de redes móviles, específicamente en el Packet Core.
