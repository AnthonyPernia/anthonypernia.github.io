Sobre mi

Soy Ingeniero Electrónico, que desde el inicio de mi carrera me intereso el desarrollo de software, actualmente estoy enfocado en el área de Big Data y Machine Learning.

He trabajado con lenguajes como Java, con mas destreza en Android, asi como tambien JavaScript , HTML, CSS en programacion web y tambien he usado programacion para hardware Arduino con librerías como Johnny-Five usando JS o el lenguaje nativo de la placa, ademas de eso, tengo mayor expertiz en Python para análisis de datos y machine learning con librerías como Pandas, Sklearn, Tensorflow, Numpy, Torch y NLTK. En cuanto a bases de datos relacionales he trabajado con PostgreSQL, MySQL, usando herramientas de Warehousing como Teradata, Hadoop y Azure. En cuanto a Infraestructura física, poseo experiencia en redes, en configuración equipos CISCO, HUAWEI y Juniper, usados en Backbone de redes móviles, específicamente en el Packet Core.
